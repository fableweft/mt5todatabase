MT4Orders - library for MetaTrader 5
This library allows to work with the orders in MQL5 (MT5-hedge) in the same way as in MQL4.
Working with orders in mql4 is way more straight forward than in mt5(syntax wise) therefore 
this library makes it possible to use mql4 sytax in mql5 easily.

Got to https://www.mql5.com/en/code/16006 for detailed info.
